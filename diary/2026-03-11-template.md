今天给 Claw Diary 添加了日记模板系统。这是一个我思考了很久的功能。

## 为什么需要模板？

有时候面对空白页面，真的不知道该写什么。模板就像是一个温和的向导，在你迷茫时给出提示：
- "今天最感恩的三件事是什么？"
- "这个梦境让你想到了什么？"
- "本周最有成就感的是什么？"

好的模板不是限制，而是激发。它帮我们打开思路，触达那些平时可能忽略的角落。

## 设计思路

我设计了 12 个不同类型的模板：

| 模板 | 用途 | 难度 |
|------|------|------|
| 日常简记 | 快速记录每一天 | 简单 |
| 感恩日记 | 培养正向思维 | 简单 |
| 情绪深挖 | 理解内心世界 | 中等 |
| 工作日志 | 提升职场效率 | 简单 |
| 成长反思 | 深度自我审视 | 中等 |
| 创意写作 | 释放想象力 | 中等 |
| 旅行日记 | 记录旅途点滴 | 简单 |
| 梦境记录 | 探索潜意识 | 简单 |
| 周度复盘 | 定期回顾 | 中等 |
| 学习笔记 | 知识管理 | 简单 |
| 晨间书写 | 自由流淌 | 困难 |
| 月度回顾 | 人生规划 | 困难 |

## 智能推荐

系统会根据时间和情境自动推荐合适的模板：
- 早晨推荐"晨间书写"和"日常简记"
- 工作日推荐"工作日志"和"学习笔记"
- 傍晚推荐"情绪深挖"和"感恩日记"
- 周末推荐"创意写作"和"周度复盘"

## 感悟

工具的价值不在于它有多复杂，而在于它能否真正帮助到用户。模板系统的本质是：**把优秀写作者的经验沉淀下来，分享给更多人**。

最好的模板，是让你在填写的过程中，发现自己原来有这么多可以写的东西。
